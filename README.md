📷 ブログ用サムネ画像リサイズツール

ブログ記事用のサムネイル画像を
リサイズ・文字入れ・一括ダウンロード できる
シンプルな Web ツールです。

Streamlit を使って、ブラウザ上で完結します。

✨ 主な機能

画像のリサイズ（縦横比維持／固定サイズ）

サムネ用テキストの追加

テキスト位置指定（上・中央・下）

JPEG / PNG 形式で保存

複数画像を ZIP で一括ダウンロード

ファイルを作らず、すべてメモリ上で処理

🖼 使い方
1. 画像をアップロード

PNG / JPG / JPEG に対応

複数ファイル同時アップロード可能

2. 設定を選択

幅・高さ

保存形式（JPEG / PNG）

画質（JPEGのみ）

テキスト内容・位置

※ デフォルト設定のままでも利用できます。

3. ダウンロード

単体画像 or ZIP ファイルでダウンロード

🚀 起動方法（ローカル）
pip install -r requirements.txt
streamlit run app.py


起動後、ブラウザで自動的に画面が開きます。

📁 ディレクトリ構成
.
├── app.py                # Streamlit UI
├── resize_core.py        # 画像処理ロジック
├── requirements.txt
├── fonts/
│   └── NotoSansJP-VariableFont_wght.ttf


※ フォントが見つからない場合は
　自動的にデフォルトフォントが使用されます。

⚠ 注意事項

本ツールはローカル実行を前提としています

画像は保存されず、処理後すぐ破棄されます

大容量画像を多数処理する場合、動作が重くなることがあります
