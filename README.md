# Thumbnail Resizer for Blog

ブログ用サムネイル画像を、  
**リサイズ・文字入れ・形式変換** できる Web アプリです。

Streamlit を使ってブラウザ上で動作します。

---

## 機能

- 画像アップロード（PNG / JPG / JPEG）
- サイズ指定でリサイズ
- 縦横比維持（ON / OFF）
- テキスト追加（中央 / 上 / 下）
- 形式変換（PNG / JPEG）
- ZIP ダウンロード対応

---

## 使い方

1. 画像をアップロード
2. 幅・高さを指定
3. 必要なら文字を入力
4. ダウンロードボタンを押す
